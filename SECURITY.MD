# Security Policy

## Supported Versions
We currently provide security updates only for the latest release.
Right now, the supported version is v7.0.0.0.

## Reporting a Vulnerability
Please report security issues privately using GitHub Security Advisories for this repository.

When reporting, include:
- A clear description of the issue and potential impact.
- Steps to reproduce (proof-of-concept if possible).
- Affected versions and environment details.

We aim to acknowledge reports within 72 hours and provide a remediation plan or status update within 7 days. Please do not open public issues for security problems.
